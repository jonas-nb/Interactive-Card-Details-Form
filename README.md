## Instalação ⚙

Instale Meu Portfólio com npm

```bash
  git clone https://github.com/jonas-nb/MeuPortfolio
  cd InteractiveCard
  npm install
  npm start
```

<br/>

## Deploy ☁

Para fazer o deploy desse projeto rode

```bash
  npm run build
```
